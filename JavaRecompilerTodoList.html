This is a list of issues that need resolution but are not being worked on yet. This does not include 'implement some 50 more missing instructions', it just contains important things to deal with when time is right, but they are important enough to be listed so they don't fall through the cracks.

  1. PUTFIELD: PutField() function similar to GetField().
  2. NEW: A similar function to invoke class initializer before creating an instance of the class.
  3. ClassInit: Create one if class does not provide it.
  4. RTTI for CHECKCAST.
