Instruction ANEWARRAY implementation

ANEWARRAY allocates one-dimensional array of objects, all references in the array are initialized with null.

Arrays in converted C++ code are implemented using a custom JavaArray class. Field 'length' is used in same way as Java array 'length' field.

The instruction extracts the array length from the operand stack, extracts array element type from the instruction and pushes 'new JavaArray<>' to the operand stack.
