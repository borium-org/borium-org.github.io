Recursive Descent Parser [Generator]

Recursive descent parsers are documented in detail in many places, starting with the usual go-to place Wikipedia. The parser I'm toying with is based on RDP by Adrian Johnstone and Elizabeth Scott. Most of RDP development happened about two decades ago, so the original ftp site is long gone but there are references to the parser in many places if you know where to look. Boost documentation reference page lists RDP among other things. Most references in there are from year 2000 or earlier, so their page is definitely not the list of most recent developments though.

So what is going on with that RDP?

The sandbox for RDP is on GitHub. It is still a work in progress, there are features that I had implemented in other repo some time ago but they are not included in this repo. Java output generator is the next step, with some of it done elsewhere.

Original RDP was self-hosting. It was capable of producing its own parser source using a BNF language grammar with embedded code in C, and there were also few modules in C that contained code that made the generated parser not just a parser but also a generator that generated itself.

Current implementation is capable of producing C output parser that is byte-for-byte same as original RDP output.

The reasoning and history of this project is documented in README.md in that GitHub repo.
