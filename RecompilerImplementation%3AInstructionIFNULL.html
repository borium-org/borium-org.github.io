Java Recompiler: Instruction Implementation

IFNULL instruction compares a reference on top of operand stack with null. If condition is satisfied, instruction at (address + offset) is executed next. Address is unsigned 16-bit location of the instruction in the bytecode array. Offset is a signed 16-bit value that is part of the instruction.

  1. Top of stack entry is popped.
  2. Type of top entry is verified to be a reference.
  3. if() statement is generated, with the body of statement generated as a goto statement to the target label. Top of stack expression is surrounded with parentheses to avoid operator precedence issues.

Recompiler will not attempt to recognize structured programming constructs such as if(), while(), etc. This is out of scope of this program. Goto was considered harmful, but the code executes just the same if goto in the bytecode is translated into goto statement in C++ code.
