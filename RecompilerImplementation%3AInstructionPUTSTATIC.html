Java Recompiler: Instruction Implementation

PUTSTATIC instruction reads the top entry from the operand stack and stores it in some object static field. PUTSTATIC is somewhat similar to PUTFIELD, with the exception that before PUTSTATIC is executed, class static initializer is invoked. The static initializer method emulates Java class loading in C++ environment.

Parameters are in stack in popping order:

  1. Value
  2. Index
