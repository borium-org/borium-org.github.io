Java Recompiler: Instruction Implementation

There are three variants of the ILOAD instruction:

  1. Constant embedded into instruction code: ILOAD 0...ILOAD 3
  2. Single byte constant: ILOAD n
  3. Double byte constant: WIDE ILOAD n

All these instructions perform the same task: Take an integer from a local variable or a parameter and push it to the stack. The only difference is in the way how the index is specified. Few conditions must be satisfied:

  1. Index is within locals range.
  2. Local at that index must be defined and must be an integer. Local variables are initially not defined, they get defined when ISTORE instructions are executed.

If these conditions are satisfied, the local variable is pushed to the operand stack.
