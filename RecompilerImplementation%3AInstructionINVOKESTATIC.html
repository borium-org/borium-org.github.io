Java Recompiler

Java Recompiler: Instruction Implementation

INVOKESTATIC is used to execute static methods. Static methods do not have 'this' pointer.

Parameters to the method are parsed using the method descriptor and they are popped from operand stack in right-to-left order.

Void static methods are handled. Code for the function call is generated, nothing is pushed to the operand stack.
