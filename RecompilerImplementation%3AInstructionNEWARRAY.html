Java Recompiler: Instruction Implementation

NEWARRAY creates an array of primitive types (not objects) and pushes the array reference to the operand stack. Primitive type is encoded by an unsigned integer in the instruction:

Array Type atype
T_BOOLEAN 4
T_CHAR 5
T_FLOAT 6
T_DOUBLE 7
T_BYTE 8
T_SHORT 9
T_INT 10
T_LONG 11

NEWARRAY is implemented using helper class JavaRawArray. The main difference between this class and JavaArray is that the data in raw array is raw, without Pointer<> wrappers.

As any other *NEW*, the new object is stored in a temporary Pointer<> wrapper for the reference so that the object is guaranteed to be destroyed if it is not assigned into any other variable.
