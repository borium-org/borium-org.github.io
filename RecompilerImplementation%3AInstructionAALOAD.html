Java Recompiler: Instruction Implementation

AALOAD instruction reads an array element and stores it on the operand stack.

AALOAD parameters are in popping order:

  1. Index
  2. Array

C++ equivalent of Java array is implemented using a pointer to JavaArray instance. Operator [] in C++ cannot be used in this context because it can't be overloaded and compiler will try to use the pointer to JavaArray as an array of pointers. Because of all this the method get() is implemented in the JavaArray to access its elements.
